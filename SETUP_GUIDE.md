## Bước 3: Cấu hình AI Provider

Bạn cần chọn 1 trong 3 AI providers và lấy API key:

### Option A: OpenAI GPT-4 (Khuyến nghị)
1. Đăng ký tại: https://platform.openai.com/
2. Tạo API key tại: https://platform.openai.com/api-keys
3. Chi phí: ~$0.01-0.03 mỗi hồ sơ

### Option B: Google Gemini Pro
1. Đăng ký tại: https://makersuite.google.com/
2. Tạo API key
3. Chi phí: ~$0.005-0.015 mỗi hồ sơ (rẻ hơn)

### Option C: Anthropic Claude
1. Đăng ký tại: https://console.anthropic.com/
2. Tạo API key
3. Chi phí: ~$0.008-0.02 mỗi hồ sơ
